# Get-IDCurrentUser

## SYNOPSIS
Get details for the current user

## SYNTAX

```
Get-IDCurrentUser [<CommonParameters>]
```

## DESCRIPTION
Returns details of the user authenticated to the current Identity session

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-IDCurrentUser
```

Get details for the authenticated user
